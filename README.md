## Explorando falhas no SSH

![hacking](https://www.sandsteinwandern.de/wandern/wp-content/uploads/2019/11/giphy-hack2.gif)

O protocolo SSH é utilizado para a comunicação remota entre dispositivos, sendo executado na porta 22. 

Neste projeto foi utilizado o ataque de força bruta com arquivos de usuários e senhas (wordlists) para encontrar credenciais.
